*** Steps to Set Up Selenium IDE ***
-> Open Microsoft Edge
-> Go to Extensions
-> Add Selenium IDE Extension

*** Steps to Record a Test ***
-> Open the Selenium IDE Extension
-> Click 'Record a new test in a new project'
-> Provide the project name
-> Provide the Base url
-> Start Recording
-> In the browser perform the actions/steps that will be recorded in the backend
-> You pause/resume/end the session from the selenium IDE
-> Once after completing the test, save and run it from the selenium IDE

*** Exporting the Recorded Test ***
-> Right click on the saved recording and export it as a code file(Ex: Java file)
